Switch Tabs
====================
一个标签页选项卡切换小效果练习

## 选项卡

- 为选项卡添加出`click`事件
- 为选项卡高亮，增加`addClass`

```javascript
	$('selected').click(function(){
		//do something...
		$(this).addClass('selected');
	});
```

## 内容区

- 全部隐藏
- 通过click获取元素的index()值来显示第n个内容区块
